LEFT AND RIGHT


DESCRIPTION

Left and Right is a Casual Arcade Game for the Android Platform.

OBJECTIVE

Try to keep the ball on the platform, by moving left or right all the way !

Just tap the screen to change the direction of the ball and keep it from falling off the platform.

How many turns can you last ?

PROJECT DETAILS

Left and Right was created using the Unity 3D Game Engine. It was programmed in the C# language.

The project has been configured to work with any Android Phone with an Operating System Version of Kitkat or Higher

The player can tap on the screen to initially start the game, and then tap the screen to change the direction of the ball,whenever required.
